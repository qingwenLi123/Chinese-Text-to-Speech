# 中文文本转语音（TTS）

一个基于 Flask + Microsoft Edge TTS 的本地 Web 工具，用于把中文文本合成为 MP3 音频。当前支持单段文本合成、多角色对话合成、环境噪音叠加、真人模式和多文件音频拼接。

## 功能

- **单段模式**：输入一段中文文本，选择音色后一键生成 MP3。
- **对话模式**：按 `姓名：内容` 的格式输入多行对话，页面会自动识别说话人，并为每个说话人配置不同音色。
- **多种中文音色**：支持普通话男声、女声、少年男声、少年女声，以及粤语、东北话、陕西话、台湾腔等音色。
- **真人模式**：在对话合成时可开启口语化处理，模拟自然停顿、口语表达和更接近真人交流的节奏。
- **环境噪音**：对话合成可叠加白噪音、粉噪音或棕噪音，并可调节噪音音量。
- **音频拼接**：上传至少 2 个 MP3 文件，支持拖拽调整顺序，并可设置片段之间的停顿时长后合并为一个文件。
- **在线播放与下载**：生成完成后可在页面中直接试听，也可以下载 MP3 文件。
- **临时文件清理**：提供接口清理超过 1 小时的临时音频文件。

## 环境要求

- Python 3.8+
- 可访问 Edge TTS 在线服务的网络环境

## 安装

```bash
pip install -r requirements.txt
```

依赖包括：

- `flask`
- `edge-tts`
- `imageio-ffmpeg`

项目通过 `imageio-ffmpeg` 自动获取 FFmpeg 可执行文件，用于音频拼接、静音填充和噪音叠加。

## 启动 Web 应用

```bash
python app.py
```

启动后在浏览器中打开：

```text
http://127.0.0.1:5000
```

## 使用说明

### 单段模式

1. 选择「单段模式」。
2. 输入要合成的中文文本。
3. 选择音色。
4. 点击「生成语音」。

单段文本长度限制为 3000 字以内。

### 对话模式

1. 选择「对话模式」。
2. 按每行 `姓名：内容` 的格式输入对话，例如：

```text
陈明：王总您好，今天想和您确认一下授信情况。
王鹏：可以，您问吧。
```

3. 在「对话人员配置」中为每个说话人选择音色。
4. 按需开启「真人模式」或配置「环境噪音」。
5. 点击「生成语音」。

对话总字数限制为 5000 字以内。生成时会按片段合成音频，并在片段之间加入短暂停顿后合并。

### 音频拼接

1. 选择「音频拼接」。
2. 上传至少 2 个 MP3 文件，最多 50 个。
3. 在文件列表中拖拽调整顺序。
4. 选择片段之间的停顿时长。
5. 点击「合并音频」。

当前仅支持 `.mp3` / `.mpeg` 文件。

## 命令行工具

项目也提供了独立的命令行 TTS 脚本 `tts.py`，支持普通话男声和女声：

```bash
# 生成女声
python tts.py "你好世界" -v female -o hello.mp3

# 生成男声
python tts.py "今天天气不错" -v male -o weather.mp3

# 生成男声、女声试听文件
python tts.py --preview
```

## 辅助脚本

- `list_voices.py`：列出 Edge TTS 中可用的中文音色。
- `test_ssml.py`：生成一组 SSML 测试音频，用于验证 Edge TTS 对停顿、语速、音调等标签的支持情况。

## 主要接口

- `GET /`：Web 页面。
- `POST /tts`：单段文本转语音。
- `POST /tts_dialogue`：多段对话合成。
- `POST /merge`：上传并合并多个 MP3 文件。
- `GET /audio/<filename>`：播放或下载生成的音频。
- `POST /cleanup`：清理过期临时音频文件。

## 文件输出

Web 应用生成的音频会保存在：

```text
static/audio/
```

生成文件主要用于临时试听和下载，建议定期调用 `/cleanup` 或手动清理不再需要的音频文件。
